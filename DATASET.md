## Schema

### Track Record

```javascript
{
  "rank": 1,                          // Position in top 10K (1-10000)
  "track": {
    "name": "Die With A Smile",       // Track title
    "id": "2plbrEY59IikOBgBGLjaoe"    // Spotify track ID
  },
  "artists": [                        // Array of contributing artists
    {
      "name": "Lady Gaga",
      "id": "1HY2Jd0NmPuamShAr6KMms"
    },
    {
      "name": "Bruno Mars",
      "id": "0du5cEVh5yTK9QJze8zA0C"
    }
  ],
  "album": {
    "name": "Die With A Smile",       // Album title
    "id": "10FLjwfpbxLmW8c25Xyc2N"   // Spotify album ID
  },
  "popularity": 100,                  // Spotify popularity score (70-100)
  "duration": "4:11",                 // Track length (M:SS)
  "archived": true,                   // Archived status on Anna's Archive
  "explicit": false,                  // Contains explicit content
  "release_date": "2024-08-16",       // Album release date (YYYY-MM-DD)
  "album_type": "single",             // album | single | compilation
  "isrc": "USUM72409273",             // International Standard Recording Code
  "copies": 9                         // Number of ISRC duplicates on Spotify
}
```

## Aggregation Methodology

Pre-computed aggregations are generated by `build-data.js`:

```javascript
// Example: By-decade breakdown
songs.forEach(song => {
  const decade = Math.floor(year / 10) * 10;
  byDecade[decade].count++;
  byDecade[decade].avgPopularity += song.popularity;
});
```

Output is stored in `js/data.js` for zero-latency runtime access.
